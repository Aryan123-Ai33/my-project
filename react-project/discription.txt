1. This component displays a product card with name, price, and status passed as properties.
function ProductCard({ name, price, status }) {
  return (
    <div className="product-card">
      <h3>{name}</h3>
      <p>Price: ${price}</p>
      <p>Status: {status}</p>
    </div>
  );
}
2. The App component holds an array of products with details like name, price, and stock status.
const products = [
  { name: "Wireless Mouse", price: 25.99, status: "In Stock" },
  { name: "Keyboard", price: 45.50, status: "Out of Stock" },
  { name: "Monitor", price: 199.99, status: "In Stock" },
];
3. It renders a header section with titles centered on the page.
<div className="header">
  <h1>Aryan Koundal</h1>
  <h1>Products List</h1>
</div>
4. The list of product cards is rendered by mapping over each product and passing its data as props to the ProductCard component.
<div className="products-row">
  {products.map((product, index) => (
    <ProductCard 
      key={index} 
      name={product.name} 
      price={product.price} 
      status={product.status} 
    />
  ))}
</div>
5. The .products-row class uses flexbox to arrange product cards horizontally in a row with spacing.
.products-row {
  display: flex;
  justify-content: center;
  gap: 30px;
}
6. Each product card is styled with a border, padding, fixed width, background color, and centered text for a neat appearance.
.product-card {
  border: 2px solid #ccc;
  border-radius: 8px;
  padding: 32px 48px;
  width: 250px;
  background: #fff;
  text-align: center;
}
7. The entire app wrapper centers the content and sets a max width for better readability.
.app-wrapper {
  margin: 0 auto;
  max-width: 1200px;
  padding-top: 30px;
}
